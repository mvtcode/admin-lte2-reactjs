# admin-lte2-reactjs

**Feature**

Admin lte2 as a web application. 

**Module**

- React
- React-router
- Redux
- Bootstrap 3
- Admin-lte2

**Install**
- npm i
- bower i

**Start**
- npm start

Application start **[http://127.0.0.1:3000](http://127.0.0.1:3000)**.

**Build**
- npm run build
